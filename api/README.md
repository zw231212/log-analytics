### 项目说明
  对外提供数据的服务项目
  ```
  以下都是按照时间来进行查询的，只有查询行为，不提供修改行为。
  1、统计浏览数、页面数、IP数、Bytes数、非浏览器数、文件数、404数、500数
  2、访问来源统计（referrer）
  3、访问时长与时间段
  4、访问者国家与城市
  5、操作系统与浏览器
  6、400与500类型的页面url
  7、访问的全部日志
  ```
  运行方式，gradle build jar，然后运行jar
